<h1 align="center">Nito</h1>

## Prerequisites

- Node.js
- MongoDB

## Getting Started

```
# clone repo
git clone https://github.com/MaximilianHagelstam/nito.git

# move into project
cd nito
```

```
# Server

# move into folder
cd server

# create .env
cp .env.example .env

# install deps
yarn

# run local mongodb instance
mongod

# start server
yarn dev
```

```
# Client

# move into folder
cd client

# create .env
cp .env.example .env

# install dependencies
yarn

# run app
yarn start
```
